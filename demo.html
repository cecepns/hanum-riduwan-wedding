📸 Demo Aplikasi Bingkai Foto & Video

Aplikasi web modern untuk menambahkan bingkai indah pada foto dan video Anda.

🚀 Coba Aplikasi Sekarang 📖 Baca Dokumentasi

✨ Fitur Utama

  • Capture Foto: Ambil foto langsung dari kamera web dengan kualitas HD
  • Rekam Video: Rekam video dengan resolusi tinggi dan kontrol penuh
  • Upload File: Upload foto/video dari perangkat dengan drag & drop
  • Bingkai Otomatis: Bingkai PNG transparan dengan output format reels (1080x1920)
  • Preview Real-time: Lihat hasil sebelum download
  • Download Kualitas Tinggi: Simpan hasil dalam format PNG/WebM
  • Progress Tracking: Monitor proses video dengan progress bar
  • Responsive Design: Bekerja sempurna di desktop dan mobile

🛠️ Teknologi yang Digunakan

HTML5

Struktur aplikasi dan elemen media modern

CSS3

Styling modern dengan gradient dan animasi

JavaScript ES6+

Logika aplikasi dengan class-based architecture

Web APIs

getUserMedia, MediaRecorder, Canvas API

📱 Cara Menggunakan

1. 📷 Mulai Kamera

Klik tombol "Mulai Kamera" dan izinkan akses kamera di browser Anda.

2. 🎯 Ambil Media

Pilih antara "Ambil Foto" atau "Mulai Video" sesuai kebutuhan Anda.

3. 🖼️ Bingkai Otomatis

Bingkai akan otomatis ditambahkan dengan format reels (1080x1920).

4. 💾 Download Hasil

Klik "Download Hasil" untuk menyimpan file dengan bingkai.
